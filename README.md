# TapDown - Social Media Video Downloader

TapDown is a minimal, high-performance web application for downloading videos from TikTok, Instagram, YouTube, and more without watermarks. Built with Next.js 15, Tailwind CSS, and Shadcn/UI.

## Features

-   **Multi-Platform Support**: Download from TikTok, Instagram, YouTube, Facebook, Twitter.
-   **No Watermarks**: Get the clean video file.
-   **Premium UI/UX**: Glassmorphism design, smooth animations, and dark mode.
-   **Mobile Reponsive**: Works perfectly on all devices.
-   **Privacy Focused**: No data storage, direct downloads.

## Tech Stack

-   **Framework**: Next.js 15 (App Router)
-   **Styling**: Tailwind CSS v4, Shadcn/UI
-   **Animations**: Framer Motion
-   **Icons**: Lucide React
-   **Backend**: Server Actions + `social-downloader-cherry`

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Deployment

### Vercel

1.  Push this repository to GitHub.
2.  Import the project in Vercel.
3.  Deploy.

No environment variables are required for the basic functionality.
